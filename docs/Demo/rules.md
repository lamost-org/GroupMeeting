## 插入图片

用法：
- `![](./figure/lamost.jpg)`  或
- `![lamost](./figure/lamost.jpg)`

效果：

![lamost](./figure/lamost.jpg)

**注意：**

- `[]` 里的内容只是针对图片的说明，可选择加或不加。
- `()` 可以用
  - 相对路径 (可以与 `.md` 文档同一级目录，也可以自己新建文件夹，但要保证路径写正确)
  - 绝对路径，只可以写绝对的网络 URL 位置，比如: 
  
`![](https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png)`
